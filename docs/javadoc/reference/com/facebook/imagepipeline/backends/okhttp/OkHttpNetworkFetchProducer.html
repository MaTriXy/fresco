Class Overview

Network fetch producer using OkHttp as a backend.

OkHttpNetworkFetchProducer supports request cancellation. This feature is enabled via cancellable constructor parameter.

For example of usage see com.facebook.fresco.sample.adapters.FrescoOkHttpAdapter
